What It Does

Searches Literature First

Uses Europe PMC by default and optional PubMed when NCBI_EMAIL is configured.

Scores Transparently

Ranks papers by gene, alteration, therapy, cancer type, response terms, and review flags.

Adds Database Support

Attaches CIViC and GDSC evidence after literature claims are found, not before.

Run Locally

The Streamlit app is dynamic Python software, so GitHub Pages hosts this static project page only.

conda env create -f environment.yml
conda activate resistance-discovery
streamlit run app.py

Research-Use Scope

This is a discovery and literature triage tool, not clinical decision support. The output should be reviewed manually before being used in scientific or translational decisions.

Built for biomarker therapy-response discovery: mutation X in gene Y, therapy Z, cancer W.
